# Do-you-trust-them❔
Do you trust them is a 3D, FPP, detective game, which the player is a office worker, who needs to find out whose the impostor in the company.
# 🔧 Technologies
- Unity
- Visual Studio 2022
- C#
# 📍 The Process
Do you trust them is a game made for my university course. This game was developed by four-person team, over the course of 4 months. I was one of two programmers in a team, responsible for implementing part of gameplay systems, integration of assets and audio. All scripts included in repository are made by me. 

I stared with the inspection system, along with the pick up ability for the player. Then I focused on printer and cabinet scripts, which allowed other the programmer to start working on days system, and implementing the clues. During integration of 3D and 2D assets, I created pasue menu. In the final stage of development I worked on audio system.
# 📚 What I Learned
- How to use reycast
- How to use conditional operator
- How to work with audio in unity
- How to switch between cameras, and have several canvases in the project
- I improved working with UI
- Creating scripts and systems with other programmers
